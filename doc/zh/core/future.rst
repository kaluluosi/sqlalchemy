SQLAlchemy 2.0 未来版本（核心）
============================

.. admonition:: 我们到了2.0了！

    此页面描述了SQLAlchemy 1.4提供的“未来”模式，目的是为了进行1.4->2.0的过渡。
    对于2.0， :func:`_sa.create_engine` 和 :class:`_orm.Session` 上的“future”参数仍然可用于向后的兼容性支持，
    但是如果指定，必须保持其值为True。
